# Cobalt Glossary

This glossary provides concise definitions of key terms and abbreviations used throughout the Cobalt SS Turbo project documentation.  These entries reference GM platform components and core concepts relevant to logging, diagnostics and tuning.

| Term | Description |
| --- | --- |
| **E69** | GM engine controller used on LNF‐era platforms |
| **LNF** | 2.0 L turbocharged, direct‐injection Ecotec engine (Cobalt SS Turbo) |
| **F35** | 5‑speed manual transaxle used in the Cobalt SS |
| **G85** | GM limited‑slip differential option for the F35 transmission |
| **DBC** | CAN database mapping frames to signals |
| **Signal Registry** | Semantic layer describing meaning, confidence and evidence for decoded signals |
| **Trace Bundle** | Packaged evidence artifact consisting of logs, decode assets and hashes |
| **Policy Engine** | Module implementing an allowlist with risk classes and preconditions enforcement |
| **Arming Window** | Time‑limited operator‑approved period for allowlisted actions |
| **Golden Trace Program** | Curated trace bundles used as regression truth |
| **Deterministic Replay** | Capability to reproduce identical outputs given the same inputs and versions |
